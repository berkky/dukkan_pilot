# DukkanPilot — Terms Template Notes

`/Terms` sayfası örnek kullanım şartları taslağıdır.

## Canlı öncesi özelleştirilecek alanlar

- Şirket ünvanı, adres, uygulama hukuku, yetkili mahkeme
- Sorumluluk sınırları ve tazminat tavanı (avukat)
- Hizmet seviyesi / bakım penceresi (SLA)
- Fesih, askıya alma, ücret iadesi (ödeme gelince)

## Ödeme / abonelik

Şu an online ödeme entegrasyonu yoktur. Ödeme eklendiğinde:

- Plan faturalama maddeleri
- Yenileme / iptal
- Vergi / fatura bilgileri

bölümleri güncellenmelidir.

## İşletme sorumluluğu

Kampanya, sadakat, menü fiyatları ve public formda toplanan müşteri verisi için işletme paneli sorumluluğu vurgulanır; sözleşmede netleştirilir.

## Süreç

1. Taslağı hukukçu ile gözden geçir
2. Placeholder’ları doldur
3. Sürüm / “son güncelleme” tarihini yayınla
4. Register/Login linklerini doğrula
