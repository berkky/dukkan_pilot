# Mobile Web Polish Checklist (Public Menu)

- [ ] Hero: işletme adı net, açıklama clamp, WhatsApp badge, logo fallback
- [ ] Kategori nav: sticky, horizontal scroll, aktif kategori state
- [ ] Ürün kartları: ad/açıklama taşmıyor, fiyat net, CTA dokunma alanı iyi
- [ ] Sepet bar: adet + toplam + “Sepeti Gör”
- [ ] Drawer: satırlar net, adet artır/azalt/sil, boş sepet state
- [ ] Pricing: ara toplam/indirim/toplam, kampanya mesajı, puan preview
- [ ] Form: label’lar, validation mesajları, kısa gizlilik notu
- [ ] Confirmation: sipariş no, toplam, WhatsApp CTA, tracking CTA
- [ ] Tracking: badge/timeline, polling error sakin, Completed/Cancelled net
- [ ] Cookie notice: CTA’ları kapatmıyor
- [ ] Legal linkler: `/Privacy` erişilebilir, minimal yer kaplıyor
- [ ] Accessibility: focus-visible, aria-current, hit targets
- [ ] Performance: lazy image, minimal JS, gereksiz tekrar yok
