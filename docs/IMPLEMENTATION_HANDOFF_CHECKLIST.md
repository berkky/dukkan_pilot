# Implementation Handoff Checklist

Satış → kurulum teslimi için işaret listesi.

- [ ] SalesRequest **Won**
- [ ] Business oluşturuldu / `BusinessId` bağlandı
- [ ] Owner hesabı hazır (login doğrulandı)
- [ ] Plan / abonelik ayarlandı
- [ ] `/Admin/Onboarding` skoru incelendi
- [ ] İşletme profili + WhatsApp
- [ ] Menü kuruldu (kategori + ürünler)
- [ ] Public menü `/m/{slug}` doğrulandı
- [ ] QR poster yazdırıldı / teslim edildi
- [ ] Test sipariş + kitchen akışı
- [ ] (Opsiyonel) kampanya veya ödül
- [ ] Personel eğitimi
- [ ] Backup / smoke (release ortamında)
- [ ] Müşteriye Kurulum Sihirbazı gösterildi
- [ ] İlk hafta takip için Customer Success notu alındı
