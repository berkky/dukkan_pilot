# Vertical Demo Packs Guide (34B)

## Amaç

Tek bir demo işletme yerine, farklı sektörlerde müşteri kendini ürünün içinde görebilsin diye **vertical demo paketleri** sunulur.

Bu içerikler gerçek müşteri verisi değildir. Seed/demo amaçlıdır.

## Public sayfalar

- Demo paket galerisi: `/DemoPacks` (alias: `/Demo/Packs`)
- Public demo menüler:
  - `/m/demo-kafe`
  - `/m/demo-tatlici`
  - `/m/demo-burgerci`
  - `/m/demo-restoran`
  - `/m/demo-nargile`

## Hangi işletmeye hangi demo?

- **Kafe**: hızlı QR menü + sadakat + kampanya auto-apply
- **Tatlıcı**: vitrin ürünler + kampanya + tekrar ziyaret
- **Burgerci**: menüler/combos + sepet artırma
- **Restoran**: kategori akışı + operasyon/mutfak hikayesi
- **Lounge**: premium vitrin + kampanya senaryosu (demo içeriktir; yerel mevzuata uyum işletme sorumluluğundadır)
