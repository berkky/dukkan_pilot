# Customer Success Playbook

## İlk gün

- Owner girişi + Kurulum Sihirbazı
- WhatsApp + menü + public link
- Bir test sipariş + kitchen demo

## İlk hafta

- QR afiş kullanımı
- Gerçek sipariş gelirse kitchen alışkanlığı
- Eksik zorunlu onboarding adımlarını kapat
- Admin: düşük skor / sipariş yok filtreleri

## İlk ay

- Kampanya veya sadakat öner
- Rapor + CRM inceleme görüşmesi
- Plan kullanımını (limit) kontrol et
- Yenileme / upgrade sinyallerini izle (SalesRequests + Billing)

## Başarı metrikleri

- Aktif ürün > 0 (ideal 5+)
- En az 1 sipariş / hafta (müşteri ölçeğine göre)
- Onboarding ReadyToLaunch veya Live
- Owner panel girişi (audit / notification aktivitesi)

## Kullanım yoksa müdahale

1. WhatsApp / ürün eksik mi? → Onboarding next action
2. Sipariş yok → birlikte test sipariş
3. Owner girmiyor → kickoff yenile / Sales handoff
4. Abonelik riski → Billing + SalesRequests

## Kampanya önerileri

- İlk sepet indirimi (min tutar)
- Sadakat ödülü (puan eşiği)
- DemoCenter üzerinden müşteriye gösterim

## Raporlama görüşmesi

- Satış / ürün / müşteri raporları
- Kampanya etkisi
- Sonraki ay hedefi

## Yenileme / upgrade sinyalleri

- Limitlere yaklaşma (ürün / kampanya)
- Aktif kullanım + düşük plan
- Won sonrası sürekli Live skor
